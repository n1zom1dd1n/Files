# Telegram Timetable Bot

## 📌 Project Description
This project is a Telegram bot that provides timetable information for university students based on their group name.

## ⚙️ Features
- Responds to /start command
- Provides timetable based on user input
- Hosted on Railway cloud platform

## 🚀 Deployment
The bot is deployed using Railway:
1. Code uploaded to GitHub
2. Connected to Railway project
3. Environment variable BOT_TOKEN added
4. Bot started using worker process

## 🧪 Testing
- Bot responds to /start command
- Bot runs 24/7 on cloud
- Verified in Telegram chat
